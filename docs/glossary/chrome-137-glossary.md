# 🧭 Chrome 137 新用語リファレンス（Glossary）

> 本Glossaryは **docs/chrome-137-2025-05-27.md** の「全体サマリ」の直後に挿入予定の内容です（PR前の準備版）。

| 用語 | 概要 | 関連領域 |
|------|------|-----------|
| **CSS `if()`** | 条件に応じて値を分岐する関数。`media()`/`supports()`/`style()` などの条件と組み合わせて使用。 | CSS |
| **Writer / Rewriter API (OT)** | オンデバイスLLMで文章生成・リライトを行う実験API。`ai.rewriter.create()` などで利用。 | AI / 実験 |
| **DevTools×Gemini** | DevTools からAI提案を受け、ワークスペースへ保存する実験機能。提案と保存を分離して運用。 | DevTools / AI |
| **PII混入** | 個人特定情報がプロンプト/ログに混ざること。入力制御と監査ログが必須。 | プライバシー |
| **SRI/CSP** | 改ざん検出と読込制御の基本。供給網攻撃に対する第一防御線。 | セキュリティ |
