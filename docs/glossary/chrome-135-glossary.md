# 🧭 Chrome 135 新用語リファレンス（Glossary）

> 本Glossaryは **docs/chrome-135-YYYY-MM-DD.md** の「全体サマリ」の直後に挿入予定の内容です（PR前の準備版／日付は既存ファイルに合わせて調整）。

| 用語 | 概要 | 関連領域 |
|------|------|-----------|
| **Scroll-driven animations** | スクロール位置に同期するアニメーション（`@scroll-timeline` 等）。パフォーマンスに優れる。 | CSS / UX |
| **Document Picture-in-Picture** | 任意の文書をPIPウィンドウとして表示。ミニプレーヤー的UIを標準で実装。 | マルチタスク / ウィンドウ |
| **File System Access** | ユーザー許可のもとローカルファイルへ読み書きするAPI群。 | ストレージ / UX |
| **COOP/COEP** | クロスオリジン分離のためのヘッダー。強い分離でSharedArrayBuffer等を有効化。 | セキュリティ / パフォーマンス |
| **Origin Trial (OT)** | 実験機能の早期評価。仕様変更のリスクと引き替えに実運用での知見を得る。 | 実験機能 |
| **Baseline** | 互換性の合意点。利用可否判断の土台。 | 互換性 |
